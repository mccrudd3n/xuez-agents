## Create Moon
In your Urbit:
```
|moon
```
Copy the key and note the moon name.

## Install New `zuse.hoon`
```
./urbit -w $MOON_NAME -G $COPIED_KEY -c $PIER_DIR
```
The moon will compile and apply OTAs. After that is done, run:
```
|mount %
```

Install files:
```
./install.sh $MOON_PIER
```
